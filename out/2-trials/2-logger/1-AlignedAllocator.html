class AlignedAllocator

A custom allocator class that aligns memory blocks to a specified size.

This class provides a way to allocate and de-allocate memory, ensuring that the allocated memory is always aligned to a specific size. It’s particularly useful for allocation of data structures with fixed-size alignment requirements, such as structures containing integers or pointers.
