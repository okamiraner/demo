# `LogBuffer` Class

A class to manage and store logs.

This class provides methods for adding, removing, and accessing logs. It is designed to be thread-safe and can handle a large number of log entries.

#### `public void `[`addLog`](#class_log_buffer_1a59788ef79a646e9e3e5ad55358782273)`(const char * logEntry)` 

Adds a new log entry to the buffer.

This method checks if the buffer has reached its maximum capacity before adding a new log entry. If it does, the oldest log entry is removed to make room for the new one.

#### Parameters
* `logEntry` The log entry to add to the buffer.

#### `public void `[`removeLog`](#class_log_buffer_1adfd35e175c7f9b95c979d9a7afac2770)`(size_t numEntries)` 

Removes a specified number of log entries from the buffer.

This method removes the oldest log entries up to a specified count, making room for new log entries if necessary.

#### Parameters
* `numEntries` The number of log entries to remove from the buffer.
