# class AlignedAllocator

Provides a memory pool with a specified alignment. It allows for aligned memory
allocation, which can be beneficial for performance in certain scenarios.

Key Features:

*   Allocates a memory pool of a specified size and alignment.
*   Provides an allocate method to allocate aligned memory.
*   Resets the allocator to the beginning of the pool with the reset method.
*   Automatically frees the allocated memory upon destruction.
