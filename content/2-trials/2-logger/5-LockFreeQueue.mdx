# `LockFreeQueue` Class

A simple logger that writes logs to a file or other output stream.

This class provides a basic way to write log messages to a file or other output stream. It includes methods for writing log messages at different levels (debug, info, warning, error).

# struct `LockFreeQueueNode` 

Node in the linked list representation of the lock-free queue.

Each node represents a single element in the queue. The node contains a value, a pointer to the next node, and any necessary metadata for concurrent access.
