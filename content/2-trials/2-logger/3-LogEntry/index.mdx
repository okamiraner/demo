# class LogEntry

The LogEntry struct represents a single log entry. It contains a timestamp,
thread ID, log level, and a message.

Key Features:
*   Uses std::chrono to capture the timestamp of the log entry.
*   Stores the thread ID of the logging thread.
*   Supports different log levels (`INFO`, `WARNING`, `ERROR`).
*   Contains a fixed-size character array for the log message.
